My Developer Journey

I have been vibe coding for a while, using curiosity and rapid prototyping to turn rough ideas into working software. I like starting with a concept, testing it quickly, learning from what breaks, and improving it until it becomes something more complete.

My interest in technology originally grew from robotics and embedded systems. I was drawn to how software can control hardware, how microcontrollers work, and how code can interact directly with the physical world.

Later, I became more invested in artificial intelligence and machine learning. That shifted my focus toward models, automation, intelligent tools, and how AI can support development workflows or become part of larger systems.

More recently, I have been moving deeper into software-based projects. I am especially interested in security-oriented utilities, basic password generators, developer tools, programming language design, package systems, and virtual operating system concepts.

Robotics & Embedded Systems Exploring microcontrollers, hardware control, low-level logic, and how software interacts with real devices.
Artificial Intelligence & Machine Learning Learning how intelligent systems work, how models can assist development, and how AI can be integrated into projects.
Software, Security & Virtual Systems Building more software-focused tools, including password utilities, custom language ideas, virtual operating systems, and developer-focused applications.
